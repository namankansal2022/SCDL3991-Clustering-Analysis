## About This Project

This repository represents the codebase for the SCDL3991 Undergraduate Research Project undertaken at the School of Biomedical Engineering, University of Sydney.

The research focuses on **data-driven phenotyping of physiological sleep signals**, developing an end-to-end machine learning pipeline encompassing signal preprocessing, feature engineering, and representation learning to identify interpretable latent structure in multimodal time-series data. The project benchmarks unsupervised clustering methods to enhance model robustness in low-label clinical settings.

> **This repository will be updated on a regular basis as the research progresses, ensuring all implementations, results, and documentation remain current and reflective of the latest stage of the project.**

---

## Current Status — Stage 2

This repository currently contains the initial exploratory data analysis and baseline implementation of standard unsupervised clustering algorithms applied to a sleep health dataset, serving as the foundation for the broader research pipeline ahead.
